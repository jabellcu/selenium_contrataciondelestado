# selenium_contrataciondelestado

This is a simple notebook to automatically add CPV codes to the
configurable alerts in contrataciondelestado.es

It requires input username and password manually.

Este es un _notebook_ sencillo para añadir códigos CPV a las alertas
(suscripciones) de portal contrataciondelestado.es

Requiere meter el nombre de usuario y contraseña manualmente.
